# GORU

A risc-v simulator 

## Pre-Requisites

1. Install [riscv](https://github.com/riscv-collab/riscv-gnu-toolchain)
2. Get [riscv-tests](https://github.com/riscv-software-src/riscv-tests)
3. install requirements

```bash
pip3 install -r requirements.txt
```


```bash
cd goru
./cpu.py
```
